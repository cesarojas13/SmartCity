# Smart City Challenge

Aplicación web interactiva para un taller de introducción a Ingeniería.

## Objetivo

Que estudiantes de colegio experimenten una simulación de ciudad inteligente usando conceptos de:

- Internet de las Cosas
- Sensores
- Automatización
- Datos
- Toma de decisiones

## Ejecutar localmente

```bash
pip install streamlit
streamlit run app.py
```

## Archivos

- `app.py`: aplicación principal.
- `requirements.txt`: dependencias para despliegue.

## Uso en taller

1. Ejecutar la app.
2. Compartir el enlace o generar un QR.
3. Cada estudiante toma decisiones para administrar la ciudad.
4. Al final, se conversa sobre cómo esa simulación se puede conectar a hardware real.
